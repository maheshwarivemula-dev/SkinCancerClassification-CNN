# Skin Cancer Classification using CNN

## Overview

A deep learning web application that classifies skin lesions using a Convolutional Neural Network (CNN) trained on the HAM10000 dataset.

## Features

* Image upload interface
* CNN-based skin lesion classification
* Confidence score prediction
* Flask backend API
* Interactive web frontend

## Dataset

HAM10000 (Human Against Machine with 10000 Training Images)

## Technologies Used

* Python
* TensorFlow / Keras
* Flask
* NumPy
* Pandas
* HTML
* CSS
* JavaScript

## Project Structure

```bash
pip install -r requirements.txt
python backend/app.py
```
